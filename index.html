How to Run

  • Open in PyCharm or VS Code
  • Run python project.py
  • Enter path: question.csv
  • Press Escape to exit fullscreen

How to Test

  • Run pytest test_project.py
  • Single test: pytest test_project.py::test_get_question
  • Requires question.csv in project root

CSV Format

  • Columns: question, choice_a … choice_d, ans
  • Minimum 10 question rows
  • Question max 45 characters
  • Answer must be A / B / C / D
